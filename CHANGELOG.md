### [v0.3.2.1] - 2026-07-15

重做外部 Agent SSH 密钥管理，收敛为按指纹组织的单卡界面。

#### 新增功能 (Added)
- **指纹账号归类**：按完整 SHA-256 指纹展示该密钥可登录的全部账号。
- **可恢复撤销**：单独撤销某个账号的公钥，并支持恢复或彻底撤销。
- **在线识别**：按认证指纹确认在线账号，展示会话数量、终端和脱敏来源。
- **权限审计**：按需查看用户组、无 sudo、受限 sudo、完整 sudo和精确允许命令；只读检查，不提供授权功能。
- **通用提示词**：提供 Agent 接管和工作机私钥彻底删除两份通用提示词，不带入开发者本机数据。

#### 安全改进 (Security)
- 专属 Agent 默认使用非 root 账号，命名可由当前用户指定或随机生成。
- 撤销、恢复和彻底撤销均按完整指纹处理，并校验并发修改、保留安全备份。
- 工作机清理先核对指纹，只卸载指定 ssh-agent 密钥并删除对应私钥文件，不影响其他密钥。
- 非 root 账号在权限检查前不再误标为普通用户；检查后区分低权限、受限运维和管理员账号。

#### 文档与清理 (Changed)
- README 改为以当前密钥管理卡为准，移除会与应用内提示词漂移的旧登记、迁移和清理命令副本。
- 工具栏统一显示“SSH 密钥管理”，删除旧本地监管模型遗留的作用域接口和未使用辅助函数。

### [v0.3.1] - 2026-06-13

引入系统托盘、窗口记忆、配置补全等功能。

####  新增功能 (Added)
- **系统托盘**：新增系统托盘图标，支持左键切换显示/隐藏，右键菜单操作。
- **隐藏全部窗口**：托盘菜单支持一键隐藏全部窗口（主窗口 + SSH 窗口）。
- **显示全部窗口**：托盘菜单支持一键恢复所有隐藏的窗口。
- **连接后自动托盘**：SSH 连接成功后自动隐藏主窗口到托盘（可在设置中关闭）。
- **窗口位置记忆**：每个窗口自动记忆位置和大小，重启后恢复到上次状态。
- **清除窗口记忆**：设置页面支持一键清除所有窗口位置记录。
- **SSH窗口关闭自动显示首页**：所有 SSH 窗口关闭后，首页自动显示并获得焦点（可在设置中关闭）。
- **配置自动补全**：升级版本后旧配置文件缺少新字段时，自动补充默认值，不丢失已有配置。
- **合并浅色请求**： [#4 feat: 新增浅色主题支持，改善强光环境下使用体验。](https://github.com/nanxiangxi/qssh/pull/4)
- **软件快捷键**：软件自身的快捷键操作，提供更舒适的体验。
- **编译补充**：完善mac，linux的编译脚本（本次不测试，仅基础完善，为后续多平台适配做准备）

#### 优化细节 (Optimize)
- **配置系统**：重构配置加载逻辑，支持增量补全缺失字段。
- **窗口管理**：SSH 窗口创建后自动显示并聚焦。
- **托盘行为**：退出时先恢复所有窗口再退出，确保正常关闭[#2 UI方面建议](https://github.com/nanxiangxi/qssh/issues/2)。
- **多行输入**：结构终端增加多行输入功能（设置命令发送方式为：发送按钮）。
- **经典终端代码高亮**：经典终端提供代码高亮选项。
- **端口转发命名**：端口转发增加命名功能 [#3 功能性优化](https://github.com/nanxiangxi/qssh/issues/3)。
- **终端菜单** 经典终端增加小菜单。

### [v0.2.0] - 2026-06-09

修复已知bug，引入全新的终端特性，配置系统，快捷键支持等。

####  新增功能 (Added)
- **结构化终端**：引入结构化终端，面向未来的终端设计，更加适配AI等特性。
- **终端模式切换**：支持结构化模式和经典模式运行时切换，数据同源。
- **命令补全**：100+ 内置命令，Tab 补全，支持命令、子命令、选项补全。
- **会话录制**：录制终端会话，导出为文本/HTML/JSON。
- **命令历史**：收藏、搜索、危险命令标记。
- **代码高亮**：支持 15+ 语言自动检测和语法高亮。
- **临时终端**：依附于当前会话的交互式终端，用于 Ctrl+R 等操作。
- **配置系统**：后端持久化存储，支持终端类型、字体大小、交互模式等配置。
- **断线重连**：自动/手动重连，提示符检测。
- **快捷键支持**：完整 Readline 键映射、TTY 控制信号、Alt/Meta 组合键。
- **多档位屏幕**：根据屏幕大小动态调整窗口初始大小。
- **版本信息**：显示软件版本号。
- **终端搜索**：快捷搜索终端的历史执行。

#### 优化细节 (Optimize)
- **AI助手**：改为使用新版本的终端
- **全局日志**：兼容新版本
- **批量命令**：兼容新版本

#### 移除功能 (Remove)
- **底部最近命令**：移除底部最近命令，改用更加合理的终端历史会话

####  修复BUG (Fix)
- **屏幕大小异常**：修复系统缩放导致的屏幕异常。
- **连接丢失**：修复断开连接后永久记录被删除的问题。
- **配置不生效**：修复配置文件实时写入问题。

### [v0.1.0] - 2026-06-05

首个测试版本正式发布！本版本奠定了系统的核心基础架构，并推出了一站式 SSH 管理控制台，为您提供从终端操作到智能运维的全链路体验。

####  新增功能 (Added)
- **️核心 SSH 模块**：首个测试版本上线，提供稳定且安全的远程连接通道。
- **️交互式终端**：支持多标签页 Web Terminal，提供流畅的命令行交互体验。
- **可视化文件管理**：内置轻量级文件浏览器，支持文件的上传、下载、编辑及目录树预览。
- **实时资源监控**：提供服务器 CPU、内存、磁盘 I/O 及网络状态的实时图表看板。
- **️防火墙管理**：支持可视化的安全组/iptables 规则配置，一键放行或拦截端口。
- **端口转发**：轻松配置本地/远程端口映射，安全穿透内网服务。
- **进程守护**：集成进程保活机制，确保关键后台任务在异常退出后自动重启。
- **AI 智能助手**：接入大语言模型，提供命令解释、报错排查及运维建议等辅助功能。
- **批量命令执行**：支持多台服务器并发下发指令，大幅提升集群运维效率。
- **全局日志审计**：集中收集并展示所有操作日志，方便回溯追踪与安全审查。
